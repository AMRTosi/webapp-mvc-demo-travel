# Arquitectura del sistema
- Estilo: Clean Architecture; capas: `Domain`, `Application`, `Infrastructure`, `Web`.
- Estándares: SOLID, DDD básico, CQRS (si aplica).
- Seguridad: autenticación OIDC (Azure AD), autorización por roles.
- Testing: xUnit/Jest; cobertura mínima 80%.
- Observabilidad: OpenTelemetry + App Insights.
